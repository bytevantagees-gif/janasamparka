## üÜò Troubleshooting

### "Still showing citizen dashboard after moderator login"

**Problem:** Login page showing old phone numbers that don't exist.

**Solution:** 
1. ‚úÖ Use the correct moderator numbers: `+9199000000X`
2. ‚úÖ Clear browser cache (Cmd+Shift+R or Ctrl+Shift+R)
3. ‚úÖ Check phone number matches exactly (including +91)

### "Department officer can't see assigned complaints"

**Problem:** Department officer login but no complaints shown.

**Solution:**
1. Ensure complaints assigned to their specific department
2. Check department_id matches in database
3. Use moderator account to assign complaints first

### "Can't create new users"

**Problem:** All test users already created.

**Solution:**
1. Use existing users from this list
2. Run: `docker compose exec backend python create_department_users.py`
3. Check database for existing users
