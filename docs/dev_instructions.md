##

\f8  Overview\
\
**Janasamparka** is a bilingual (Kannada + English) mobile and web platform connecting **citizens, MLAs, and departments** for grievance redressal, project transparency, and rural empowerment.\
\
The system includes:\
- **Citizen App** (Flutter)\
- **MLA/Admin Web Dashboard** (ReactJS)\
- **Backend APIs** (FastAPI + PostgreSQL + PostGIS)\
- **AI + Speech + Map Integrations**\
- **Secure Cloud Deployment (HPE / ByteVantage Cloud / AWS)**\
